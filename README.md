# Nimbo House

Storefront for **Nimbo House**, an online shop specializing in handcrafted wall resin art.

## Products

- **Geode Paintings** — resin art inspired by natural geode formations
- **Beach Art** — coastal-themed resin pieces
- **Resin Trees** — tree-inspired wall art crafted in resin

## Tech Stack

- **Platform:** Shopify
- **Theme:** Horizon (customized)
